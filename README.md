# Smart Door Lock System using ESP32

## Overview
This project uses RFID authentication and ESP32 to control an electronic door lock securely.

## Features
- RFID-Based Authentication
- Automatic Door Unlocking
- Access Control
- Smart Security System

## Components
- ESP32
- RC522 RFID Reader
- Relay Module
- Solenoid Lock

## Working
When a valid RFID card is scanned, ESP32 activates a relay that unlocks the door for a few seconds and then automatically locks it again.
